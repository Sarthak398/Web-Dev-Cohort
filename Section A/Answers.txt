Ques : What problem does a Version Control System solve?
Ans : It tracks changes in files, supports collaboration, and enables version rollback.

Ques : Differentiate between Centralized and Distributed VCS.
Ans  : Centralized VCS relies on a central server, while Distributed VCS lets every user have a full repository copy.

Ques : What is Git and how is it different from GitHub?

Ques : What does the .git folder contain?

Ques : What is git branching and command to create new branch and switching from one to another branch
Ans  : Git branching allows parallel development; use git branch new-branch and git checkout new-branch.
 
Ques : What is the difference between git reset and git reset --hard?
Ans  : git reset moves HEAD but keeps changes; git reset --hard discards all changes.

Ques : What is a squash merge, and when would you use it?
Ans  : A squash merge combines commits into one, used to simplify history before merging.

Ques : Explain the use of git rebase with an example scenario.
Ans  : git rebase rewrites commits for cleaner history, e.g., updating a feature branch with the latest main changes.
